# leadit-web

회사 홈페이지 리뉴얼 프로젝트를 포트폴리오용으로 정리한 저장소입니다.

## 프로젝트 개요
기존 홈페이지의 화면 구성을 개선하고, 사용자에게 필요한 정보를 더 직관적으로 전달할 수 있도록 리뉴얼한 웹 프로젝트입니다.

## 담당 업무
- 메인 및 서브 페이지 화면 구현
- 공통 UI 스타일 정리
- 페이지 레이아웃 및 인터랙션 작업
- 반응형 웹 대응

## 기술 스택
- Next.js
- TypeScript
- CSS Module

## 안내
본 저장소는 포트폴리오 용도로 정리한 버전이며, 불필요한 빌드 파일은 제외했습니다.
